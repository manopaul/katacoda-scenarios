Directory Commands

dir Briefly list directory contents
`dir`{{execute}}


pwd Print Working Directory
`pwd`{{execute}}


cd Change Directory
`cd /`{{execute}}
--> pwd Print Working Directory


mkdir Create new folder(s)
`mkdir myFolder`{{execute}}


--> cd into the folder you created
`cd myFolder`{{execute}}


rmdir Remove folder(s)
`cd ../`{{execute}}


`rmdir myFolder`{{execute}}
